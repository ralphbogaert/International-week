# Goalkeeper Project

Project scaffold for goalkeeper progression modeling.

## Structure

- data/raw: Original source files and competition folders
- data/interim: Temporary intermediate datasets
- data/processed: Final model-ready datasets
- notebooks: Exploration and experiments
- src: Reusable source code modules
- outputs/tables: Output tables
- outputs/models: Trained model artifacts
- outputs/visuals: Figures and charts

## Getting Started

1. Create and activate a Python virtual environment.
2. Install dependencies with:

   pip install -r requirements.txt

3. Run the starter entrypoint:

   python src/main.py
